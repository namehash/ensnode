:::note
This guide covers running ENSApi locally for development and contributions.
:::

## Configure Environment Variables

```bash
cd apps/ensapi
cp .env.local.example .env.local
```

:::note
ENSApi's `.env.local` should be placed at `apps/ensapi/.env.local`, _not_ at the monorepo root.
:::

<LinkCard
  title="ENSApi Configuration Options"
  href="/ensapi/usage/configuration"
/>

## Running ENSApi

```bash
# from monorepo root
pnpm run -F ensapi dev

# from apps/ensapi
pnpm run dev
```
