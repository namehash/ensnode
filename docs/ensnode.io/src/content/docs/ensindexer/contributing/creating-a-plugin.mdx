:::note
This guide covers how to create a plugin for ENSIndexer.
:::

1. add new enum value to `PluignName` `packages/ensnode-utils/src/types.ts`
2. add required datasource to config in `apps/ensindexer/src/plugins/index.ts`
3. create `plugins/NAME/NAME.plugin.ts` based on previous
  - optionally include `plugins/NAME/handlers/*.ts`
4. add your plugin to `apps/ensindexer/ponder.config.ts` `ALL_PLUGINS`
