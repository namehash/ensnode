This page uses [`enssdk`](/docs/integrate/integration-options/enssdk) with the [ENS Omnigraph API](/docs/integrate/omnigraph) to load **owner**, **resolver contract**, and **registration expiry** for a name (default: `nick.eth`). The [Quickstart](/docs/integrate) shows the raw **Domain By Name** GraphQL from `@ensnode/ensnode-sdk`; here you can edit and run a typed client snippet against a real ENSNode endpoint.
