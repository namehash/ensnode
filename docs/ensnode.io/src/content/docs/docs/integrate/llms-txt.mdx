This documentation site publishes its content in the [`llms.txt`](https://llmstxt.org/) format so AI agents and LLMs can load it directly as context.

:::tip[Prefer `ensskills`]
The recommended way to give your agent ENS capabilities is [AI/LLM Tooling](/docs/integrate/ai-llm) — purpose-built [`ensskills`](/docs/integrate/integration-options/ensskills) that teach your agent how to drive ENSNode, and that already instruct it to read this documentation site via `llms.txt` when it needs more. Reach for the raw endpoints below only if you aren't using `ensskills`.

## Available endpoints

- [`/llms.txt`](https://ensnode.io/llms.txt) — a structured index of the documentation with links to every page.
- [`/llms-full.txt`](https://ensnode.io/llms-full.txt) — the entire documentation concatenated into a single file, ready to drop into a model's context window.

## Use it with your agent

Paste this at the top of a prompt to point your agent at the full documentation before asking your question:

```md title="prompt.md"
Load the ENSNode documentation from https://ensnode.io/llms-full.txt to answer the following question:
```
