ENSApi is the REST API service that provides access to ENS data indexed by ENSNode.

## Features

- **Name resolution** — resolve ENS names to records, addresses, and content hashes
- **Primary names** — look up primary names for Ethereum addresses
- **Name token browsing** — explore registered ENS name tokens
- **Registrar actions** — query registration, renewal, and transfer history

## Quick Example

Resolve records for an ENS name:

```bash
curl https://api.alpha.ensnode.io/api/resolve/records/nick.eth
```
